How to copy paste in GNU nano?
Copy/Paste/Cut Text in Nano
To copy the marked text press Alt + ^.
To cut the marked text press ^K (Ctrl +K).
To paste the marked text, move the cursor to a suitable position and press ^U (Ctrl + U).

https://www.youtube.com/watch?v=VDAwbgHoYEA - Creating a CSV in Android Studio
-Steps:
1. Add a button that will call a method when clicked
2. Inside xml directory in res, create provider_paths.xml and put in appropriate code (from video)
3. Add this path to the AndroidManifest.xml file using the provider tag
4. Generate (or collect) data in MainActivity.java
5. Save this data internally to a CSV file
6. Make an intent that takes the file and exports it
